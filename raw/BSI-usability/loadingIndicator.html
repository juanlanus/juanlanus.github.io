Loading indicator

During the "load" time, in addition to displaying something atop the form, change the cursor pointer to the combined arrow with an hourglass named progress.

body { cursor:progress; }

This is the standard way to communicate the user that she can do things while there is more data is on its way to the screen.

This is needed because the user might (will) not be looking at the Loading ... banner a the top of the screen, but focused on her task in a specific location somewhere else.

But, no matter where on the screen is she working, she will always be seenig the pointer no matter if she has noticed the Loading ... banner or not.

The browser automatically set this cursor pointer in some cases, while the page is loading, but it might fail to do so during XHR acrions.
