Use checkboxes for binary data

In a properties edition form a binary input was implemented using a dropdown. edition sub-form the situation is even worse.

This must be a checkbox.

This is exactly the use case for a checkbox: a yes or no situation.

In addition to the advantage of being operated with a single mouse click, a checkbox conveys a message of binary option while a dropdown suggests that there are additional options that should be checked.

The right implementation would be thus:

The "Enabled" label is actually an HTML <label> control. This has to be done this way for accesibility, but is also benefitial for all users because when the label and the checkbox are related this way clicking either makes the checkbox change.

According to the Fitt's law, bigger mouse targets are captured faster improving application efficiency, which is a component of usability.

The accesibility need stems from the fact that a screen reader can´t tell the relationship between a label and it's associated control.
