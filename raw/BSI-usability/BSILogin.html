BSI Usability

BSI Login

This login screen must open with focus in the first Username field to allow the user to start writing right away. It opens with focus in none of the form controls.

This forces the user to do a mouse action before starting to write.

Also, the "Login" button must be connected to the Enter key, so if the user's browser is pre-filling the Username and Password fields, hitting Enter takes the user into the application with a single keystroke.

Else, she has to grab the mouse to set the focus or click "Login".
