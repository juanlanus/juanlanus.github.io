un color

número Identificación del color para pedidos, por ejemplo "44F"
grupo número de fila de muestras 1, 2, 3, ... donde aparece
orden ordenamiento en su fila de muestras, número no necesariamente consecutivo
archivo imagen nombre del archivo, sin path, ej: "44f.jpg"
descripción un texto que describa el color y la superficie
oscuro o claro "oscuro" o "claro", para determinar el color del texto que va sobre la muestra, ej: el n° del color negro debe ir en blanco, y viceversa
comentarios cualquier información adicional que el cliente debería ver en el momento de elegir un color, como limitaciones en las medidas, plazos de entrega especiales, etc...

Se almacenan en un Object MDatosDeLosColores que es en realidad un array asociativo. La clave es, por ejemplo, color_1 para el color número "1", color_44F para el color "44F", etc.

Cada color es un array de 7 ítems (0 ... 6) que tienen significado:

  1. numero número del color "1"
  2. grupo grupo de colores donde se muestra 1
  3. orden orden en la gana de colores (y en su grupo) 1
  4. imagen nombre del archivo con la foto "1.jpg"
  5. nombre nombre del color " "
  6. tono si es claro u oscuro, para rotularlo en negro o blanco "claro"
  7. comentario comentarios adicionales, un texto posiblemente extenso " "
  8. elegido es true si este color está elegido en una sesión

Los nombres asociados a cada ítem son constantes para usar como subscripts al hacer referencia a los datos individuales del color.
