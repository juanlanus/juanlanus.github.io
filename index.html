    Working prototypes created to showcase interactions.

      The requirement was to allow the users to select items from a rather lengthy set where the structure of the data was known to be hierarchical.

      The use case was about managing a big numbers of columns of a grid, allowing to select and reorder them.

      This is a small website for an acrylic sheets factory. Its edge is the greater colors offering.

    • Contextual help:
      A wireframe showcasing a set of recommendations on how to implement contextual help in a portal-like huge application so it's consistent accross all pages, also allowing to decouple and reuse the help development effort.
  • Applications
    Complete applications that I designed and developed.

      My client operates a complex system for recording and processing millions of traffic offenses for big argentine province.
